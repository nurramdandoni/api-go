# api-go
Learning go with Fiber

Install Go Compiler
### https://go.dev/

Install Fiber
> go get github.com/gofiber/fiber/v2

Install Jwt for go
> go get github.com/dgrijalva/jwt-go

Run Project
> go run hey.go

Buka Browser/ Postman
> http://localhost:3000

Route dengan Autorization
> http://localhost:3000/auth/cekAuth
